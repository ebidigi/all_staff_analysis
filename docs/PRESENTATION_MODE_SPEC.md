# プレゼンモード 要件定義書

## 概要

定例ミーティングでお客様に見せるための、シンプルで見やすいプレゼンテーション専用画面。
案件ごとに表示項目をカスタマイズ可能。

---

## アクセス方法

- **URL**: `presentation.html`（独立したページ）
- **リンク**: 月内管理ダッシュボードのヘッダーに「プレゼンモード」ボタンを追加

## 機能要件

### 1. 案件選択

- ドロップダウンで案件を選択
- 選択した案件のデータのみを表示
- 案件名は画面上部に大きく表示

### 2. 期間選択（月間比較）

- 「比較元」月と「比較先」月を選択可能
- 例: 「2024年9月」 vs 「2024年8月」
- デフォルト: 当月 vs 前月

### 3. 表示項目の切り替え

設定パネル（折りたたみ可能）から以下を選択:

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| 架電数 | ON | 実績数 |
| PR数 | ON | 実績数 |
| アポ数 | ON | 実績数 |
| 架電toPR率 | ON | 率指標 |
| PRtoアポ率 | ON | 率指標 |
| 架電toアポ率 | ON | 率指標 |

### 4. 前月比較の表示形式

各指標に以下を表示:
- **当月の数値**（大きく）
- **増減数**: +100件 / -50件
- **増減率**: (+20%) / (-10%)
- 率指標の場合は **ポイント差**: +2.1pt / -1.3pt

### 5. 担当者別内訳（オプション）

- デフォルトは非表示
- ボタンクリックで展開/折りたたみ
- 表形式で担当者ごとの実績を表示
- 選択した表示項目のみを列として表示

### 6. 設定パネル

- 画面上部に配置
- 折りたたみ可能（プレゼン中は非表示にできる）
- 含まれる設定:
  - 案件選択
  - 比較月選択（2つのドロップダウン）
  - 表示項目チェックボックス

## UI/UXデザイン

### カラー

- 背景: 白または薄いグレー（プロジェクターで見やすく）
- 増加: 青色（#1155cc）
- 減少: 赤色（#e04f24）
- テキスト: 濃いグレー（#434547）

### フォント

- 案件名: 太字、24-32px
- 数値: 太字、36-48px
- 増減: 16-20px
- ラベル: 14-16px

### レスポンシブ

- 最小幅: 1024px（プロジェクター表示想定）
- カードは横並びを基本
- 担当者テーブルは横スクロール可能

---

## データ仕様

### 入力データ

既存のTurso DBから取得:
- `performance_rawdata`: 実績データ
- `employees`: 社員マスター

### 計算ロジック

```
架電toPR率 = PR数 / 架電数 × 100
PRtoアポ率 = アポ数 / PR数 × 100
架電toアポ率 = アポ数 / 架電数 × 100

増減数 = 比較元月の値 - 比較先月の値
増減率 = (比較元月の値 - 比較先月の値) / 比較先月の値 × 100
ポイント差 = 比較元月の率 - 比較先月の率
```
